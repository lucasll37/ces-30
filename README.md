# Projeto Final da Disciplina CES-30 Computação Gráfica

## Ambiente
- #### Node.js `v18.18.0`
- #### npm `v10.2.1`

## Modo de Uso

- ### instalação das dependências
```bash
npm install
```

- ### Build do projeto (somente necessário para produção)
```bash
npm run build
```	
- ### Execução do projeto
```bash
npm run dev
```

- ### Deploy do projeto (somente para produção)
```bash
npm run deploy
```
